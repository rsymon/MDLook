# MDLook Roadmap

MDLook is intentionally a reader, not an editor. Possible follow-up work:

- Theme switching with a small set of local reading themes.
- GitHub-flavored Markdown behavior review.
- Obsidian-style wiki link handling.
- Stronger local image path support inside sandbox limits.
- Mermaid support feasibility study.
- Table of contents support for long Markdown files.
- GitHub Release packaging.
- DMG packaging and notarization workflow.
- Smaller settings UI focused on reading preferences only.
